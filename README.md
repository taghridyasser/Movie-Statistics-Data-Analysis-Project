## Overview
This project performs statistical analysis on a movie dataset, including cleaning, handling missing values, and generating visualizations. It also applies statistical methods from the `scipy` library.

## Dataset
- **Source:** `movie_statistic_dataset.csv`
- Contains attributes such as movie titles, release years, ratings, and other statistical metrics.

## Features
- Load and inspect movie data.
- Handle missing values and clean the dataset.
- Generate descriptive statistics.
- Create visualizations of trends and distributions.
- Apply statistical functions from `scipy`.

## Technologies Used
- Python 3
- pandas
- matplotlib
- scipy
